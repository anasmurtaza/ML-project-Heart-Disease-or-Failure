# ❤️ Heart Failure Prediction Using Machine Learning

A Machine Learning project for predicting the risk of heart failure
using clinical patient data. This project applies supervised learning
algorithms to classify whether a patient is at risk of heart failure
based on medical features.

------------------------------------------------------------------------

# 📌 Project Overview

Heart failure is a serious cardiovascular disease that can lead to
severe complications if not detected early. This project uses machine
learning techniques to analyze patient records and predict heart
failure, helping support early diagnosis and clinical decision-making.

------------------------------------------------------------------------

# 🎯 Objectives

-   Predict heart failure using Machine Learning.
-   Analyze important medical features.
-   Compare classification algorithms.
-   Evaluate model performance.
-   Identify the best-performing model.

------------------------------------------------------------------------

# 📂 Dataset

The dataset contains clinical records of heart failure patients.

### Features

-   Age
-   Anaemia
-   Creatinine Phosphokinase (CPK)
-   Diabetes
-   Ejection Fraction
-   High Blood Pressure
-   Platelets
-   Serum Creatinine
-   Serum Sodium
-   Sex
-   Smoking
-   Time

### Target

-   0 → Patient Survived
-   1 → Death Event (Heart Failure)
